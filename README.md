## ğŸ› ï¸ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Navigate to the project directory
cd payglobe-website

# Install dependencies
npm install

# Run development server
npm run dev
```

### Build for Production

```bash
npm run build

```
